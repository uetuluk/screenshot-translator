# Screenshot Translator

Translates text in screenshots to other English and transliterates to Latin script.

## Installation

```bash
npm install
```

### Environment Variables

```bash
cp .env.example .env
```

Fill in the environment variables in `.env` file.

Fill ANTHROPIC_API_KEY.

## Usage

```bash
npm build
npm start
```

## Development 
```bash 
npm run dev
```
